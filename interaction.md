# 交互设计方案

## 核心交互功能
1. **多步骤债务评估流程**
   - 步骤1：品牌展示和引导
   - 步骤2：债务类型选择（银行/非银）
   - 步骤3：化债场景选择（核心/周边）
   - 步骤4：智能计算器
   - 步骤5：对接留资

2. **智能计算器**
   - 银行债务核心化债计算
   - 非银债务核心化债计算
   - 实时结果显示
   - 多种还款方案对比

3. **动态内容切换**
   - 根据选择的债务类型显示不同选项
   - 根据选择的场景显示不同计算器
   - 平滑的页面过渡动画

## 用户体验流程
用户进入网站 → 查看品牌介绍 → 选择债务类型 → 选择化债场景 → 使用计算器 → 获取对接方案
